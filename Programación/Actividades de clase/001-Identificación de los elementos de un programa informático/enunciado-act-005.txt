Contexto

En el mundo digital, los videojuegos han ganado una popularidad increíble. Uno de los juegos más populares es League of Legends (LoL), un juego de estrategia en equipo que requiere mucha práctica y dedicación para dominar. Además, muchos amantes del café disfrutan de actividades creativas como el arte latte, donde se mezcla café con leche para crear patrones impresionantes.
Enunciado paso a paso

    Declaración de constantes:
        Declara una constante llamada PI y asígnale el valor 3.1415.
        Imprime el valor de PI.

    Modificación del valor de una variable:
        Declara una variable llamada radio y asígnale un valor inicial.
        Calcula el área del círculo utilizando la fórmula ( \text{área} = \pi \times \text{radio}^2 ).
        Imprime el área del círculo.

    Aplicación práctica:
        Considera que estás jugando a LoL y tienes un personaje con cierto número de vidas.
        Declara una variable llamada vidas y asígnale un valor inicial.
        Crea una función llamada recibirDaño que reste 1 vida a la variable vidas.
        Llama a esta función varias veces y imprime el número de vidas restantes.

    Arte latte:
        Declara una constante llamada COLOR_MARRON con un valor que represente el color marrón en hexadecimal.
        Declara una variable llamada patron y asígnale un valor inicial (por ejemplo, "espuma").
        Crea una función llamada dibujarPatron que imprima el patrón de arte latte utilizando la variable patron.
        Llama a esta función varias veces con diferentes patrones.

Restricciones

    Las constantes deben formularse con mayúsculas.
    Las variables deben formularse con minúsculas.
    No uses valores mágicos en el código.

Criterios de evaluación

    Introducción y contextualización (25%):
        Explica cómo se utiliza la constante PI en cálculos matemáticos.
        Describe cómo las variables son utilizadas para almacenar datos que cambian durante la ejecución del programa.

    Desarrollo técnico correcto y preciso (25%):
        Verifica que las constantes estén declaradas correctamente con mayúsculas.
        Confirma que las variables estén declaradas correctamente con minúsculas.
        Asegúrate de que el cálculo del área del círculo sea preciso.

    Aplicación práctica con ejemplo claro (25%):
        Comprueba que la función recibirDaño funcione correctamente y reste las vidas adecuadamente.
        Confirma que la función dibujarPatron imprima patrones de arte latte correctamente.

    Cierre/Conclusión enlazando con la unidad (25%):
        Reflexiona sobre cómo el uso de constantes y variables puede facilitar el mantenimiento del código.
        Relaciona los conceptos aprendidos con las actividades creativas como el arte latte y el juego de LoL.
