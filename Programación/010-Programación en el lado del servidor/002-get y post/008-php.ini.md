php.ini en Windows:
C:/xampp/php/php.ini
(Espero que no sea vuestro caso)

En Ubuntu está en:
/etc/php/8.3/apache2/php.ini

Si no os deja modificar con gedit

sudo nano /etc/php/8.3/apache2/php.ini

Comandos:
(Parece que sí que deja hacer scroll)
Ctrl + W = Buscar (where)
Ctrl + O = Guardar
Ctrl + X = Salir
Tenéis la ayuda bajo del editor

display errors = On (cambiad de Off a On)

Cuando se toca el php.ini hay que reiniciar apache

sudo service apache2 restart
(para aplicar los cambios)
