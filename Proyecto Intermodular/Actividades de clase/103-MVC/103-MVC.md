---
En este ejercicio se desarrolla un sistema web básico utilizando Flask, junto con el manejo de archivos JSON y plantillas HTML. 

Flask permite crear aplicaciones web dinámicas en Python de forma sencilla, gestionando la lógica del servidor y la interacción con el usuario a través del navegador. Por su parte, JSON se utiliza para almacenar datos estructurados de manera ligera y legible, mientras que HTML define la presentación visual de dichos datos en la interfaz. 

El propósito del ejercicio es aplicar los conceptos de programación web mediante la lectura de datos desde un archivo JSON y su aplicación en una plantilla HTML a través de Flask. De esta manera, se busca fortalecer la comprensión del flujo de datos entre la capa lógica y la capa de presentación, así como la integración de tecnologías intermodulares en el desarrollo web.

---
El proyecto consta de tres archivos principales:

1. En el archivo curriculum.json, que contiene los datos personales en formato estructurado: se define un objeto con los datos personales, organizados en campos como nombre, apellidos, correo electrónico, teléfono, dirección y código postal. Este formato permite una lectura sencilla y directa desde Python mediante la librería estándar json.
```
{
	"datos personales":{
		"nombre":"Fabiana",
		"apellidos":"Sotillo",
		"email":"fsotillocuevas@gmail.com",
		"telefono":"663966940",
		"direccion":"calle de fabiana",
		"código_postal":"46000 Valencia"
	},
	"experiencia":"",
	"formación":""

}
```

2. En el archivo curriculum.py, se importa el módulo Flask y la función render_template_string, además del módulo json para manejar el archivo de datos. A continuación, se crea una instancia de la aplicación ```Flask con app = Flask(__name__)```.
Dentro de la función cv(), que se asocia a la ruta raíz ("/"), se abre el archivo curriculum.json y se carga su contenido en la variable data. Luego se accede al diccionario datos personales y se guarda en dp. 
```
from flask import Flask, render_template_string
import json

app = Flask(__name__)

@app.route("/")
def cv():
	with open("curriculum.json",encoding="utf-8") as f:
		data = json.load(f)
	dp = data["datos personales"]
	
	with open("curriculum.html", encoding="utf-8") as f:
		html = f.read()
		
	return render_template_string(html, **dp)
		
if __name__ == "__main__":
	app.run(debug=True)
```

2. Posteriormente, se lee el archivo curriculum.html como texto y se renderiza utilizando render_template_string, pasando los valores de dp como variables para que puedan mostrarse en la plantilla visual donde se muestran los datos.
```
<!doctype html>
<html lang="es">
	<head>

		<style>
			html{background:grey;font-family:sans-serif;}

```

Finalmente, la aplicación se ejecuta con app.run(debug=True), lo que permite acceder al sitio web localmente en la dirección http://127.0.0.1:5000/.
